# Keymaps

Leader: `Space`

## Navigation

- `<Space><Space>` ‚Äî Find files
- `<Space>/` ‚Äî Live grep in project
- `<Space>b` ‚Äî Open buffers
- `<Space>.` ‚Äî Recent files
- `<Space>,` ‚Äî Resume last `fzf-lua` picker
- `<Space>p` ‚Äî Recent projects
- `<Space>ss` ‚Äî Symbols in current file
- `<Space>sS` ‚Äî Symbols in workspace
- `<Space>e` ‚Äî Toggle file tree
- `<S-h>` / `<S-l>` ‚Äî Previous / next file tab
- `<Space>bd` ‚Äî Close current file buffer
- `<C-h>` / `<C-j>` / `<C-k>` / `<C-l>` ‚Äî Move between windows
- `<Space>v` ‚Äî Vertical split
- `<Space>s` ‚Äî Horizontal split

## Terminal

- `<Space>t` ‚Äî Toggle terminal split
- `<Esc>` ‚Äî Leave terminal mode
- `<C-h>` / `<C-j>` / `<C-k>` / `<C-l>` ‚Äî Move between windows from terminal mode

## LSP

- `gd` ‚Äî Go to definition
- `gi` ‚Äî Go to implementations
- `gr` ‚Äî Find references / usages
- `K` ‚Äî Hover documentation
- `<Space>rn` ‚Äî Rename symbol
- `<Space>ca` ‚Äî Code action
- `<Space>fo` ‚Äî Format buffer
- `<Space>ds` ‚Äî Show diagnostics under cursor
- `[d` / `]d` ‚Äî Previous / next diagnostic
- `<Space>xx` ‚Äî Workspace diagnostics

## Git

- `]h` / `[h` ‚Äî Next / previous git hunk
- `<Space>hp` ‚Äî Preview hunk
- `<Space>hs` ‚Äî Stage hunk
- `<Space>hr` ‚Äî Reset hunk
- `<Space>gb` ‚Äî Toggle git blame view for the current file

## Editing

- `<A-d>` ‚Äî Duplicate current line below
- `<A-j>` / `<A-k>` ‚Äî Move current line down / up
- Visual `<A-j>` / `<A-k>` ‚Äî Move selected block down / up

## Selection

- `v` ‚Äî Characterwise visual mode
- `V` ‚Äî Linewise visual mode
- `<C-v>` ‚Äî Block visual mode
- `viw` / `vaw` ‚Äî Select inner word / word with surrounding space
- `vip` / `vap` ‚Äî Select inner paragraph / paragraph
- `vi(` / `va(` ‚Äî Select inside parentheses / around parentheses
- `vi{` / `va{` ‚Äî Select inside braces / around braces
- `vi[` / `va[` ‚Äî Select inside brackets / around brackets
- `vi"` / `va"` ‚Äî Select inside double quotes / around double quotes
- `vi'` / `va'` ‚Äî Select inside single quotes / around single quotes
- `vit` / `vat` ‚Äî Select inside HTML/XML tag / around tag
- `vaf` / `vif` ‚Äî Select outer / inner function
- `vac` / `vic` ‚Äî Select outer / inner class or struct

## Folding

- `za` ‚Äî Toggle fold under cursor
- `zc` ‚Äî Close fold under cursor
- `zo` ‚Äî Open fold under cursor
- `zM` ‚Äî Close all folds
- `zR` ‚Äî Open all folds

## Small Built-ins

- `r<char>` ‚Äî Replace the character under cursor
- `x` ‚Äî Delete the character under cursor
- `s` ‚Äî Delete character and enter insert mode
- `cw` ‚Äî Change from cursor to end of word
- `ciw` ‚Äî Change inner word
- `C` ‚Äî Change to end of line
